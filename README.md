# ResinPulse

Epoksi reçine atölyesi operasyon yönetim platformu — istasyon envanteri, döküm partileri, ekipman bakımı, kürlenme kontrolleri ve kalıp siparişleri.

## Demo

## Geliştirme

```bash
docker compose up
```

- Frontend: http://localhost:3018
- Backend API: http://localhost:4018/api
- Health: http://localhost:4018/api/health
